# Tucson Restaurant Analysis (Yelp Data)

This project analyzes Yelp business data to identify restaurants in Tucson
with the longest average daily opening hours.

## What this project does
- Loads Yelp business dataset
- Filters Tucson restaurants
- Parses opening hours
- Computes average daily open time
- Ranks restaurants by availability

## Tech used
- Python
- Pandas
- Yelp Open Dataset
